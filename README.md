# Sistema de Gestión de Ventas

## 📌 Descripción

Sistema web desarrollado para administrar productos, clientes y ventas, permitiendo llevar un control de stock de forma simple y eficiente.

---

## 🎯 Objetivo del proyecto

Este proyecto surge como evolución de un trabajo práctico grupal realizado durante la cursada.
Fue rediseñado y desarrollado de manera individual con el objetivo de mejorar la estructura, aplicar buenas prácticas y consolidar conocimientos en desarrollo backend y bases de datos.

---

## ⚙️ Tecnologías utilizadas

* C#
* ASP.NET Web Forms
* SQL Server
* ADO.NET

---

## 🧩 Funcionalidades principales

* 🔐 Login de usuarios
* 📦 Gestión de productos (alta, modificación, baja y listado)
* 👥 Gestión de clientes
* 🛒 Registro de ventas con múltiples productos
* 📉 Control de stock automático
* 📄 Visualización de detalle de ventas

---

## 🧱 Arquitectura del proyecto

El sistema está organizado en capas para una mejor separación de responsabilidades:

* **Web** → Interfaz de usuario
* **Negocio** → Lógica de negocio y validaciones
* **Dominio** → Entidades del sistema
* **Data** → Acceso a base de datos

---

## 🗄️ Base de datos

La base de datos permite gestionar la información de:

* Productos
* Clientes
* Ventas
* Detalles de venta
* Usuarios

Incluye relaciones entre entidades para garantizar integridad de datos y consistencia en las operaciones.

---

## 🚀 Cómo ejecutar el proyecto

1. Clonar el repositorio
2. Crear la base de datos en SQL Server
3. Ejecutar los scripts de creación de tablas
4. Configurar la cadena de conexión en el archivo de configuración
5. Ejecutar el proyecto desde Visual Studio
